# Cyclic Queue in F#

Academic project developed as part of coursework in functional programming.

## Description

This project implements a cyclic queue (circular queue) data structure in F#.

The implementation includes:

- Queue creation
- Enqueue operations
- Dequeue operations
- State management
- Validation of queue behaviour
- Example usage through script files

## Technologies

- F#
- .NET

## Purpose

The purpose of the project was to practice data structures, functional programming concepts, and implementation of queue operations in F#.
